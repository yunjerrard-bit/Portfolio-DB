</objective>

<task id="1">
<title>sheet_per_ticker.py — *_median/_std 컬럼 hidden 적용</title>

<behavior>
write_sheet_for_ticker 함수 말미(set_column 기본 너비 호출 직후)에:

```
for col_idx, col_name in enumerate(layout):
    if col_name.endswith(("_median", "_std")):
        ws.set_column(col_idx, col_idx, None, None, {"hidden": True})
```

기존 `ws.set_column(0, len(layout) - 1, 12)` 호출은 유지하되 그 다음에 위 hidden 루프 추가. (`set_column`은 같은 컬럼에 두 번 호출하면 마지막이 우선)
</behavior>
