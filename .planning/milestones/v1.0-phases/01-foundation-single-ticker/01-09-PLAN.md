<title>build_column_layout — DIFF 루프 순서 교체</title>

<behavior>
sheet_per_ticker.py의 build_column_layout 내부 DIFF 섹션을 다음과 같이 변경:

이전:
```
for price in _PRICES:
    for n in _EMA_PERIODS:
        base = f"DIFF_{price}_{n}"
        layout += [base, f"{base}_median", f"{base}_std"]
```

이후:
```
for n in _EMA_PERIODS:
    for price in _PRICES:
        base = f"DIFF_{price}_{n}"
        layout += [base, f"{base}_median", f"{base}_std"]
```

다른 그룹(OHLCV, EMA_Close, dailychg, Stoch/RSI) 루프는 그대로.
</behavior>
