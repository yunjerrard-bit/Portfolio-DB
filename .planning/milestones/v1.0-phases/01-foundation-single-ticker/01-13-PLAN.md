```python
def add_pct_change_columns(df: pd.DataFrame) -> pd.DataFrame:
    out = df.copy()
    out["Close_pct_change"] = out["Close"].pct_change()
    out["Volume_pct_change"] = out["Volume"].pct_change()
    return out
```

add_expanding_stats가 일반 DATA_COLS 리스트로 동작한다면 DATA_COLS에 `Volume_pct_change`를 포함시키고 Volume은 제외 (Volume_median/_std 더 이상 사용 안 함). expanding median/std는 Volume_pct_change 시리즈에서 계산되어 컬럼 `Volume_pct_change_median`, `Volume_pct_change_std`로 출력.

기존 Volume_median, Volume_std 출력 컬럼은 더 이상 생성하지 않음 (DATA_COLS에서 Volume 제거).
