<peg_two_step_contract>
<!-- PER 대칭 설계 — 실행자가 구현할 공개 2단계 계약 -->
PER(2단계):  matrix["EPS_ttm"][q] (분모 셀, compute_matrix 산출) → price_ratio(cell, price) → PER value
PEG(2단계):  ① 가격 주입으로 PER value 확보(위 price_ratio 결과)
             ② eps_ttm = matrix["EPS_ttm"][q].value (per-share EPS, 최근 TTM)
                eps_prior = matrix["EPS_ttm"][q-4].value (4분기 전 per-share EPS — _calendar_quarter_offset(q,-4))
             ③ compute_peg_cell(per_value, eps_ttm, eps_prior) → PEG MetricCell
PEG는 PER(가격 의존)에 다시 의존 → registry/compute_matrix가 값 직접 산출 불가(D-07 가격 비결합).
따라서 호출자(Phase 9/10)가 가격 주입 후 위 3단을 호출. compute_peg_cell은 그 공개 진입점.
</peg_two_step_contract>
